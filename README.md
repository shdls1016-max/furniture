# FORME Furniture Store Skin

흰색과 라이트 그레이를 기반으로 제작한 카페24용 가구 쇼핑몰 스킨 디자인입니다.

## 페이지

- `index.html` — 메인
- `products.html` — 상품 목록
- `product-detail.html` — 상품 상세
- `cart.html` — 장바구니
- `login.html`, `signup.html` — 회원 화면
- `notice.html`, `qna.html`, `review.html` — 게시판

모든 CSS는 각 HTML의 `<head><style>` 안에 있으며, 상품·메뉴 카드도 HTML에 직접 작성되어 있습니다. 구매, 결제, 로그인 같은 실제 동작은 카페24 모듈 연결을 전제로 한 디자인 시안입니다.
